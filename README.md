# 서론
- Node.js 및 Express 모듈을 이용한 수강신청사이트 제작
- 충남대학교 전파정보통신공학과 4학년 1학기 <네트워크 프로그래밍> 기말 프로젝트 2020.06

# 기술 스택
- Back-End: Node.js
- Front-End: HTML/CSS, javascript

# 주요 기능
- 로그인 / 로그아웃
- 사용자 추가
- 사용자 정보 수정
- 수강 신청 / 취소
- 강의 개설
- 로깅

# 특징
- '수강 신청'이라는 주제로 단순 CRUD로 구현됨
- 사용자 로그인/로그아웃 기록을 로그에 저장 (winston 모듈 이용)
